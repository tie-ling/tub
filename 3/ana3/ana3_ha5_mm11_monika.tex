\begin{Satz}[Beppo Levi]
  Ist
  \((f_{n})_{n \in \mathbb{N}}\colon \Omega \to \interval{0}{\infty}\) eine
  monoton wachsende Folge von messbaren
  Funktionen, so gilt
  \[\lim_{n \to \infty}{\int_{\Omega}{f_{n} \dd \mu}} =
    \int_{\Omega}{\lim_{n \to \infty}{f_{n} \dd \mu}}\]
\end{Satz}
